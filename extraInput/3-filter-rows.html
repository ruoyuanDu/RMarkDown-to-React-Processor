Filter Rows of a Dataset

The filter() function is used to select a subset of rows (i.e., observations) from a dataset that meets a specified condition. Commonly used filtration operations include bigger than >, bigger or equal >=, smaller than <, smaller or equal <=, equal ==, not equal !=, logical AND &, logical OR |, logical NOT !, multiple condition %in%, etc.

Filter by a single criterion

library(dplyr)
starwars2 <- starwars[, 1:5]

# get beings with height lower than 100
starwars2 %>% filter(height < 100)
  # A tibble: 7 × 5
    name                  height  mass hair_color skin_color

  2 R5-D4                     97    32 <NA>       white, red 
  3 Yoda                      66    17 white      green      
  4 Wicket Systri Warrick     88    20 brown      brown      
  # ℹ 3 more rows
# get beings with blond hair; 
# use "==" to judge if two quantities are equal
starwars2 %>% filter(hair_color == "blond")

You can use functions to create filtration criteria.

# select beings whose "mass" contains missing value NA
# use is.na() to check if a value is NA
starwars2 %>% filter(is.na(mass))
  # A tibble: 28 × 5
    name           height  mass hair_color   skin_color
    <chr>           <int> <dbl> <chr>        <chr>     
  1 Wilhuff Tarkin    180    NA auburn, grey fair      
  2 Mon Mothma        150    NA auburn       fair      
  3 Arvel Crynyd       NA    NA brown        fair      
  4 Finis Valorum     170    NA blond        fair      
  # ℹ 24 more rows
# get beings taller than the global average of all beings
starwars2 %>% filter(height > mean(height, na.rm = T))
  # A tibble: 52 × 5
    name              height  mass hair_color    skin_color
    <chr>              <int> <dbl> <chr>         <chr>     
  1 Darth Vader          202   136 none          white     
  2 Owen Lars            178   120 brown, grey   light     
  3 Biggs Darklighter    183    84 black         light     
  4 Obi-Wan Kenobi       182    77 auburn, white fair      
  # ℹ 48 more rows
# get beings whose hair color has "grey" component
library(stringr)
starwars2 %>% filter(str_detect(hair_color, "grey"))
  # A tibble: 3 × 5
    name           height  mass hair_color   skin_color
    <chr>           <int> <dbl> <chr>        <chr>     
  1 Owen Lars         178   120 brown, grey  light     
  2 Wilhuff Tarkin    180    NA auburn, grey fair      
  3 Palpatine         170    75 grey         pale

The above example uses str_detect() from popular stringr package to check if a vector of strings contain “grey”.

Filter by multiple criteria

# get beings that has NA value in "mass" column, AND "height" below 100
# use "&" for logical AND (intersection)
starwars2 %>% filter(is.na(mass) & height < 100 )
  # A tibble: 1 × 5
    name   height  mass hair_color skin_color 
    <chr>   <int> <dbl> <chr>      <chr>      
  1 R4-P17     96    NA none       silver, red
# get beings either with black hair, OR with mass below 50
# use | for logical OR (union)
starwars2 %>% filter(hair_color == "black" | mass < 50)
  # A tibble: 24 × 5
    name              height  mass hair_color skin_color 
    <chr>              <int> <dbl> <chr>      <chr>      
  1 R2-D2                 96    32 <NA>       white, blue
  2 Leia Organa          150    49 brown      light      
  3 R5-D4                 97    32 <NA>       white, red 
  4 Biggs Darklighter    183    84 black      light      
  # ℹ 20 more rows
# get beings with hair color either blond OR black, OR white
# use %in% to specify a scope of matches
starwars2 %>% filter(hair_color %in% c("blond", "black", "white"))
  # A tibble: 20 × 5
    name              height  mass hair_color skin_color
    <chr>              <int> <dbl> <chr>      <chr>     
  1 Luke Skywalker       172    77 blond      fair      
  2 Biggs Darklighter    183    84 black      light     
  3 Anakin Skywalker     188    84 blond      fair      
  4 Yoda                  66    17 white      green     
  # ℹ 16 more rows
