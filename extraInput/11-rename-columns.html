Rename Columns of a Dataset

Rename columns with rename()

rename() alters the names of selected columns by employing the syntax: new_name = old_name.

library(dplyr)
starwars2 <- starwars[, 1:5]

# use "H" in replace of the old name "hair_color" 
# use "S" in replace of the old name "skin_color" 
starwars2 %>% rename(H = hair_color, S = skin_color)
  # A tibble: 87 × 5
    name           height  mass H     S          
    <chr>           <int> <dbl> <chr> <chr>      
  1 Luke Skywalker    172    77 blond fair       
  2 C-3PO             167    75 <NA>  gold       
  3 R2-D2              96    32 <NA>  white, blue
  4 Darth Vader       202   136 none  white      
  # ℹ 83 more rows

Alternatively, you can use a named vector inside all_of() to specify the update of column names.

x1 <- c(H = "hair_color", S = "skin_color")
starwars2 %>% rename(all_of(x1))
  # A tibble: 87 × 5
    name           height  mass H     S          
    <chr>           <int> <dbl> <chr> <chr>      
  1 Luke Skywalker    172    77 blond fair       
  2 C-3PO             167    75 <NA>  gold       
  3 R2-D2              96    32 <NA>  white, blue
  4 Darth Vader       202   136 none  white      
  # ℹ 83 more rows

Note that if the named vector contains an element that is not one of the column names in the input dataset (e.g., “country” in the example below), it’ll incur an error. To increase the tolerance to such inconsistency, use any_of() instead.

x2 <- c(H = "hair_color", S = "skin_color", C = "country")

# "country" is not a column name in "starwars2"
starwars2 %>% rename(any_of(x2))
  # A tibble: 87 × 5
    name           height  mass H     S          
    <chr>           <int> <dbl> <chr> <chr>      
  1 Luke Skywalker    172    77 blond fair       
  2 C-3PO             167    75 <NA>  gold       
  3 R2-D2              96    32 <NA>  white, blue
  4 Darth Vader       202   136 none  white      
  # ℹ 83 more rows

Use functions to rename by rename_with()

Use rename_with() to rename columns with a function that specifies the renaming rule. The function takes the 2nd argument (.fn), and columns to be renamed takes the 3rd argument (.cols). If this 3rd argument is not specified, the function defaults to be applied to all columns.

# capitalize all column names
starwars2 %>% rename_with(.fn = toupper)
  # A tibble: 87 × 5
    NAME           HEIGHT  MASS HAIR_COLOR SKIN_COLOR 
    <chr>           <int> <dbl> <chr>      <chr>      
  1 Luke Skywalker    172    77 blond      fair       
  2 C-3PO             167    75 <NA>       gold       
  3 R2-D2              96    32 <NA>       white, blue
  4 Darth Vader       202   136 none       white      
  # ℹ 83 more rows

The code above is equivalent to:

# the function defaults to apply to every column (everything)
starwars2 %>% rename_with(.fn = toupper, .cols = everything())

Alternatively, in the 3rd argument, you can indicate specific columns to be renamed. (The argument names are omitted.)

starwars2 %>% rename_with(toupper, c(hair_color, skin_color))
  # A tibble: 87 × 5
    name           height  mass HAIR_COLOR SKIN_COLOR 
    <chr>           <int> <dbl> <chr>      <chr>      
  1 Luke Skywalker    172    77 blond      fair       
  2 C-3PO             167    75 <NA>       gold       
  3 R2-D2              96    32 <NA>       white, blue
  4 Darth Vader       202   136 none       white      
  # ℹ 83 more rows

You can use the purrr style and an anonymous function (marked by the tilde ~) to rename the columns. For instance, here for each selected column (noted by .x), you can replace the underscore _ with a hyphen - in the column names. Use ends_with() to select columns whose names are ended with “color”.

starwars2 %>% rename_with(
  ~ stringr::str_replace(.x, pattern = "_", replacement = "-"),
  ends_with("color"))
  # A tibble: 87 × 5
    name           height  mass `hair-color` `skin-color`
    <chr>           <int> <dbl> <chr>        <chr>       
  1 Luke Skywalker    172    77 blond        fair        
  2 C-3PO             167    75 <NA>         gold        
  3 R2-D2              96    32 <NA>         white, blue 
  4 Darth Vader       202   136 none         white       
  # ℹ 83 more rows
