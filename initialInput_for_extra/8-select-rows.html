Select Rows of a dataset

You have learned filter() to select rows based on a specified criteria. The family of functions slice_*() is another useful tool for row selection. In this tutorial, you’ll learn:

  • slice(): select rows by their integer index.
  • slice_head() and slice_tail(): select the first or last rows.
  • slice_min() and slice_max(): select rows with the smallest or largest values of a variable.
  • slice_sample(): randomly selects rows.

Subset rows with slice()

Use slice() and integer row indices to select rows. Use positive integers to include rows, and negative integers to remove rows.

library(dplyr)
starwars2 <- starwars[, 1:5]

# select the 3rd row
starwars2 %>% slice(3)
  # A tibble: 1 × 5
    name  height  mass hair_color skin_color 
    <chr>  <int> <dbl> <chr>      <chr>      
  1 R2-D2     96    32 <NA>       white, blue
# select the 3rd to 5th row
starwars2 %>% slice(3:5)
  # A tibble: 3 × 5
    name        height  mass hair_color skin_color 
    <chr>        <int> <dbl> <chr>      <chr>      
  1 R2-D2           96    32 <NA>       white, blue
  2 Darth Vader    202   136 none       white      
  3 Leia Organa    150    49 brown      light
# drop the first four rows
starwars2 %>% slice(-(1:4))
  # A tibble: 83 × 5
    name               height  mass hair_color  skin_color
    <chr>               <int> <dbl> <chr>       <chr>     
  1 Leia Organa           150    49 brown       light     
  2 Owen Lars             178   120 brown, grey light     
  3 Beru Whitesun Lars    165    75 brown       light     
  4 R5-D4                  97    32 <NA>        white, red
  # ℹ 79 more rows

Extended helper functions

Use slice_head() to select the first n rows, or prop (proportion) of rows of the dataset. In like manner, use slice_tail() to select rows from the end of the dataset.

# select the top 3 rows
starwars2 %>% slice_head(n = 3)
  # A tibble: 3 × 5
    name           height  mass hair_color skin_color 
    <chr>           <int> <dbl> <chr>      <chr>      
  1 Luke Skywalker    172    77 blond      fair       
  2 C-3PO             167    75 <NA>       gold       
  3 R2-D2              96    32 <NA>       white, blue
# select the first 10% of the rows
starwars2 %>% slice_head(prop = .1)
  # A tibble: 8 × 5
    name           height  mass hair_color skin_color 
    <chr>           <int> <dbl> <chr>      <chr>      
  1 Luke Skywalker    172    77 blond      fair       
  2 C-3PO             167    75 <NA>       gold       
  3 R2-D2              96    32 <NA>       white, blue
  4 Darth Vader       202   136 none       white      
  # ℹ 4 more rows

When the dataset has grouping variables (i.e., specified by group_by()), the slicing will be applied respectively to each group. This applies to slice() and all the slice_* helper functions (as in many other dplyr functions, but not in arrange()).

# select the first 2 rows within each group of "hair_color"
starwars2 %>% 
  group_by(hair_color) %>% 
  slice_head(n = 2)
  # A tibble: 18 × 5
  # Groups:   hair_color [12]
    name              height  mass hair_color    skin_color
    <chr>              <int> <dbl> <chr>         <chr>     
  1 Mon Mothma           150    NA auburn        fair      
  2 Wilhuff Tarkin       180    NA auburn, grey  fair      
  3 Obi-Wan Kenobi       182    77 auburn, white fair      
  4 Biggs Darklighter    183    84 black         light     
  # ℹ 14 more rows

Note that if n is greater than the number of rows in each group, the output will be the actual group size. The number of rows calculated based on proportion will be rounded down to an integer. For instance, consider the following dataset.

x <- tibble(
  class = rep(c("A", "B", "C"), c(1, 2, 3)),
  sales = rnorm(6, 500, 50))
x
  # A tibble: 6 × 2
    class sales
    <chr> <dbl>
  1 A      463.
  2 B      522.
  3 B      561.
  4 C      472.
  5 C      561.
  6 C      527.

The code below selects 50% of rows from each level of “class”, or 0.5, 1 and 1.5 rows from A, B, and C, respectively. With a rounding-down rule, 0, 1, and 1 row is taken separately from each group and retained in the output dataset (information of class “A” is thus missed).

x %>% group_by(class) %>% slice_head(prop = .5)
  # A tibble: 2 × 2
  # Groups:   class [2]
    class sales
    <chr> <dbl>
  1 B      522.
  2 C      472.

Use slice_min() to select rows containing the minimum values of a specified variable. Alternatively, use slice_max() to select rows containing the maximum values of a variable. Consider the following example.

x <- tibble(class = letters[1:6],
            A = c(10, 3, 8, 8, 8, 5),
            B = c(2, 3, 8, 6, 5, 10))
x
  # A tibble: 6 × 3
    class     A     B
    <chr> <dbl> <dbl>
  1 a        10     2
  2 b         3     3
  3 c         8     8
  4 d         8     6
  5 e         8     5
  6 f         5    10
# select 3 rows containing the minimum values of "B" column
x %>% slice_min(B, n = 3)
  # A tibble: 3 × 3
    class     A     B
    <chr> <dbl> <dbl>
  1 a        10     2
  2 b         3     3
  3 e         8     5

In the presence of ties, more rows than requested will be returned. In the following example, the output dataset has a total of four rows instead of requested two rows, as three rows contain the tied value 8 of variable “A”.

# select 2 rows containing the maximum values of "A"
x %>% slice_max(A, n = 2)
  # A tibble: 4 × 3
    class     A     B
    <chr> <dbl> <dbl>
  1 a        10     2
  2 c         8     8
  3 d         8     6
  4 e         8     5

Use with_ties = FALSE to return exactly n matches in presence of ties.

# select 2 rows containing the maximum values of "A"
x %>% slice_max(A, n = 2, with_ties = F)
  # A tibble: 2 × 3
    class     A     B
    <chr> <dbl> <dbl>
  1 a        10     2
  2 c         8     8

When selecting multiple variables, wrap the variable names inside tibble(). In the example below, rows are first selected based on the 1st variable “A”; in presence of ties in “A”, rows are then selected from the second variables “B”.

x %>% slice_max(tibble(A, B), n = 4)
  # A tibble: 4 × 3
    class     A     B
    <chr> <dbl> <dbl>
  1 a        10     2
  2 c         8     8
  3 d         8     6
  4 e         8     5

Use slice_sample() to randomly select n rows, or a prop (proportion) of rows.

starwars2 <- starwars[, 1:5]
starwars2 %>% slice_sample(n = 3)
  # A tibble: 3 × 5
    name           height  mass hair_color skin_color
    <chr>           <int> <dbl> <chr>      <chr>     
  1 Taun We           213    NA none       grey      
  2 Luke Skywalker    172    77 blond      fair      
  3 Dormé             165    NA brown      light

Use replace = T for sampling with replacement.

starwars2 %>% slice_sample(n = 3, replace = T)
  # A tibble: 3 × 5
    name         height  mass hair_color skin_color 
    <chr>         <int> <dbl> <chr>      <chr>      
  1 Boba Fett       183  78.2 black      fair       
  2 Gregar Typho    185  85   black      dark       
  3 Wat Tambor      193  48   none       green, grey
